# SCTec--cursoDevSoft-Desafio_Pratico
 SCTec | carreira DevSoft | Desafio Prático
README

esse projeto é uma landing page / site de apresentação pessoal contendo alguas informações necessárias para atingir au meta:
a. apresentação pessoal
b. competências e experiência prévias ao estudo atual em tecnologia
c. alguns projetos, reais simples (mas não tive tempo de colocar o link),em vez de fictícios mais complexos.

Há 4 arquivos, este incluso:    index.html
                                styles.css
                                script.js                
cada qual construindo respectivamente sua parte do alicerce do site.


1. HTML: estruturado em
    HEADER com barra de menu e o título / nome do estudante
    MAIN com 4 seções subdivididas em DIVs, organizadas conforme a necessidade do conteúdo. são estas:
        DESCRIÇÃO PESSOAL: sobre mim, techs utilizadas e em aprendizagem, e soft skills (p/ o ambiente corporativo)
        FORMAÇÃO & COMPETÊNCIAS: tanto atualmente quanto prévias. envolve desde histórico do ensino superior a idiomas e hobbies.
        PROJETOS: apresentados com cards/boxes para os quais não tive tempo de desenvolver o background de lógica JS mais a fundo. minha intenção - frustrada - era de criar popup na tela aumentando no viewport o conteúdo do projeto selecionado. tive que interromper esete trabalho por falta de prazo.
        CONTATO: redes sociais e dados para contato
        e concluindo com um FOOTER

2. CSS:
trabalho intenso de:
grid template
flexbox
responsividade conforme device vistos nas aulas aplicada ao conteúdo e ao menu "hamburger"

3. Javascript:
. manipulação de DOM com: getElementById, createElement, querySelector
. criação e uso de variáveis CONST e LET, arrow function, , classList, addEventListener, forEach(), appendChild()
. modificação do CSS pelo JS com: "myLet.style.___ = ''"

o projeto ficou pronto, espero que esteja a altura do programa, mas o entrego incompleto por não ter tido tempo de pesquisar tudo que eu queria implemetar em JS e entregar-lhes um site melhor.
